## Установка и запуск / Installation and launch

### Веб-версия (Web)

Нажмите **Run game** прямо на странице itch.io. Игра запустится в браузере. Требуется подключение к интернету.

> Click **Run game** on the itch.io page. The game will launch in your browser. Internet connection is required.

---

### Windows — установщик (`Nemesis-0.1.0-x64.exe`)

1. Скачайте `Nemesis-0.1.0-x64.exe`.
2. Запустите файл.
3. Следуйте инструкциям установщика.
4. Запустите игру через ярлык на рабочем столе или в меню Пуск.
5. Требуется подключение к интернету.

> 1. Download `Nemesis-0.1.0-x64.exe`.
> 2. Run the installer.
> 3. Follow the setup steps.
> 4. Launch the game from the desktop shortcut or Start menu.
> 5. Internet connection is required.

---

### Windows — портативная версия (`Nemesis-0.1.0-portable.exe`)

1. Скачайте `Nemesis-0.1.0-portable.exe`.
2. Поместите файл в любую папку.
3. Запустите двойным кликом.
4. Требуется подключение к интернету.

> 1. Download `Nemesis-0.1.0-portable.exe`.
> 2. Place it in any folder.
> 3. Double-click to run.
> 4. Internet connection is required.

---

### Linux (`Nemesis-0.1.0-x64.AppImage`)

1. Скачайте `Nemesis-0.1.0-x64.AppImage`.
2. Сделайте файл исполняемым:
   ```bash
   chmod +x Nemesis-0.1.0-x64.AppImage
   ```
3. Запустите:
   ```bash
   ./Nemesis-0.1.0-x64.AppImage
   ```
4. Требуется подключение к интернету.

> 1. Download `Nemesis-0.1.0-x64.AppImage`.
> 2. Make it executable:
>    ```bash
>    chmod +x Nemesis-0.1.0-x64.AppImage
>    ```
> 3. Run:
>    ```bash
>    ./Nemesis-0.1.0-x64.AppImage
>    ```
> 4. Internet connection is required.

---

## Требования / System requirements

- **OS:** Windows 10/11, Linux, или современный браузер
- **RAM:** 4 GB
- **Storage:** 200 MB
- **Internet:** обязательно / required
- **Input:** мышь и клавиатура / mouse and keyboard
